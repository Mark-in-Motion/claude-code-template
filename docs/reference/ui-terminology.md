# UI Terminology

This file is the shared vocabulary for user-facing labels, terms, and naming conventions. Keep it current so product copy, design, code, and documentation all describe the same things the same way.

## How to update terms

- Add a new term when a user-facing concept is introduced
- Rename an existing term by updating the definition and every place it appears
- Retire an old term by marking it deprecated and replacing it everywhere visible
